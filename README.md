# ramblings

A personal blog for technical writing, book reviews, and thinking out loud.

**[ramblings.cgym.dev](https://ramblings.cgym.dev)**

## Setup

```bash
bun install
```

## Development

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Stack

React, TypeScript, Vite, Tailwind CSS, React Router
